# Qt自定义虚拟键盘代码

## 简介
本仓库提供了一个模拟Qt插件虚拟键盘的资源文件，包括数字键盘和全键盘，支持大小写切换和中英文输入，适用于Windows和Linux操作系统。

## 功能特点
- **数字键盘**：提供基本的数字输入功能。
- **全键盘**：支持字母、数字和符号的输入，支持大小写切换。
- **中英文输入**：支持中文和英文输入，满足不同语言环境的需求。
- **跨平台**：代码经过测试，可在Windows和Linux系统上正常运行。
